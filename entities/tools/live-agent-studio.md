# Live Agent Studio

The Live Agent Studio (hosted under the oTTomator brand) is Cole's own open-source platform for building, hosting, and showcasing AI agents as a community. He describes it in personal terms at [0:02:13]: "this is the live agent Studio the platform that I've been pouring my heart and soul into over the past few months to build up this place for us to collaborate on open-Source AI agents together." The defining property, repeated across nearly every video that mentions it, is that it is community-driven and open-source first, as he restates at [0:01:21]: "the live agent studio is an incredible community-driven platform."

For a builder, the Studio's pitch is that it hands you the whole surrounding application so you only write the agent. At [0:09:12] Cole spells out the leverage: "if you integrate it with the studio you instantly have a full front end with chat and conversation history super robust." That means an agent built to the Studio's contract is portable and immediately usable, which is why he says at [0:05:41] "I built the live agent Studio to show you how to best build an agent that you can use anywhere." The platform is framework-agnostic on the agent side. In the DeepSeek V3 build he confirms a Pydantic AI agent is "compatible with the automator live agent Studio" at [0:06:53], and elsewhere he hosts the exact agent from a video there for viewers to try with free tokens ([0:08:14]).

The Studio evolved publicly. Cole built it to pre-release in roughly two months, noting at [0:00:17] "it is called the live agent studio and it is now in pre-release," then kept refining it over "the last 3 to four months" ([0:04:29]). Its biggest community moment was a hackathon: at [0:10:30] he announces "my platform automator is hosting an AI agent hackathon competition in partnership with voice flow and n8n and the price pool is $5,000," where the entry requirement was simply to "build an AI agent for the automator live agent studio" ([0:00:19]). The Studio is thus both a product and a teaching vehicle: a real place to ship agents and the through-line of Cole's agent-building tutorials.

## Related

- The Studio is the canonical showcase for [Community-Driven Development](../../concepts/community-driven-development.md) and a full [Agent Development Lifecycle](../../concepts/agent-development-lifecycle.md) from prototype to hosted agent.
- It rewards [Rapid Prototyping](../../concepts/rapid-prototyping.md) and [Agentic Coding](../../concepts/agentic-coding.md) by supplying the front end, chat, and history so builders focus on the agent and its [Context Engineering](../../concepts/context-engineering.md).
- Being framework-agnostic, it pairs with any [Model Selection](../../concepts/model-selection.md) and hosts agents built with [Pydantic AI](./pydantic-ai.md) on models like [DeepSeek V3](./deepseek-v3.md).
- Hackathon partners: [Voiceflow](./voiceflow.md), [n8n](./n8n.md).
