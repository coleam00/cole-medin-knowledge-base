# Organizations

The companies and research labs behind the tools and ideas.

- [Anthropic](anthropic.md) - Maker of Claude and Claude Code, creator of MCP, and Cole's pick for the best coding LLM and the lab whose research most shapes his practice.
- [Bright Data](bright-data.md) - Video sponsor; provides an MCP server plus web scrapers and proxies that let agents access the internet like a human, solving captchas and platform-specific scraping.
- [Cartesia](cartesia.md) - Named as one of the swappable speech providers available for the LiveKit voice pipeline.
- [Cohere](cohere.md) - LLM provider added to oTToDev via a community integration; its rerank models are the go-to reranker in Cole's n8n RAG pipelines.
- [CTO.ai](cto-ai.md) - Company founded by Lior Weinstein, partnering on the AI transformation workshop about restructuring organizations for AI.
- [DeepSeek](deepseek.md) - Chinese startup whose R1 open-source model was tied to Nvidia's $600B one-day drop; later shipped DeepSeek 3.2.
- [DigitalOcean](digital-ocean.md) - Cloud host mentioned as an example of running n8n in the cloud, where webhook endpoints must be protected.
- [Dynamous](dynamous.md) - Cole's community for early AI adopters, home of the Agentic Coding course, weekly workshops, and the shared Context Engineering Hub.
- [Google](google.md) - Author of the A2UI generative-UI specification used in the demo.
- [Groq](groq.md) - A fast inference provider Cole recommends (with a Llama model) when you need speed over raw power.
- [Hugging Face](hugging-face.md) - AI platform whose blog post and AI Agents course provide the starting definition: programs where LLM outputs control the workflow to achieve a goal.
- [Meta AI](meta.md) - Cited as an example of a model trained with ~10x fewer parameters than GPT, pointing toward faster/cheaper training.
- [Nvidia](nvidia.md) - Co-sponsor of the AI agents developer contest whose technologies Cole must integrate to qualify.
- [OpenAI](openai.md) - Maker of GPT-4o mini (affordable-but-strong chat model) and text-embedding-3-small (recommended RAG embedding model).
- [oTTomator](ottomator.md) - Cole's earlier open-source AI community (ottomator.ai), pitched as a one-stop shop for AI tools, networking, and knowledge; home of the Live Agent Studio.
- [Scrimba](scrimba.md) - Interactive coding-education platform where tutorials become pair-programming sessions you can edit in real time; a recurring sponsor of Cole's videos.
- [Shopify](shopify.md) - E-commerce platform whose CEO Toby Lütke framed context engineering vs. prompt engineering (a framing Karpathy endorsed); also built its own AI workflow engine, Roast.
- [Snyk](snyk.md) - Developer-security company trusted for securing AI-generated code; provides an MCP server for automatic vulnerability detection and sponsors webinars on securing vibe coding.
- [Sonar](sonar.md) - Code-quality and security company behind SonarQube; promotes an agent-centric development lifecycle (guide, verify, solve) and hosts Sonar Summit, where Cole gives a fireside chat on self-validation and guardrails.
- [StackBlitz](stackblitz.md) - Maker of the commercial bolt.new; partnered with the open-source bolt.diy community, bringing prize-backed contribution events and hosting the repo under its GitHub org.
- [Stripe](stripe.md) - Payments company that is Cole's headline example of an AI coding harness at scale (1,300 AI-generated PRs per week via Stripe Minions); also the external payments API agents integrate and call.
- [StrongDM](strongdm.md) - Company that went viral for running a dark factory shipping thousands of lines of production code no human writes or reviews; source of the hold-out pattern Cole adopts for reliability.
- [Vercel](vercel.md) - Frontend deployment and cloud platform (Next.js, v0) that is Cole's favorite deploy target, integrated for automatic deployment via MCP; also cited for its production-AI infrastructure and text-to-SQL agent research.
- [Weaviate](weaviate.md) - Open-source vector database platform whose article and diagrams Cole uses to teach naive/traditional RAG versus agentic RAG.
