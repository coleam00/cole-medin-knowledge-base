# I Created the BEST AI Agent Platform in Just 2 MONTHS!

Cole introduces the [Live Agent Studio](../entities/tools/live-agent-studio.md), a platform he built over two months as part of the [oTTomator](../entities/organizations/ottomator.md) (automator) community. Its differentiator versus existing agent showcases is that it combines four things no other platform does together: every agent is open source, the community can contribute agents ([community-driven development](../concepts/community-driven-development.md)), agents are standardized so they can be built with any framework ([provider independence](../concepts/provider-independence.md)), and every agent ships with full chat and conversation history in a ChatGPT-like [chat interface](../concepts/chat-interfaces.md). Users get 100 free tokens to try agents directly in the browser, with a token system covering LLM and API infrastructure costs.

The platform launches with nine agents, some contributed by other creators (a business-idea researcher and a LinkedIn/X/blog content creator), establishing the community-driven model from day one. Cole stresses framework independence: agents can be built with [n8n](../entities/tools/n8n.md), [Pydantic AI](../entities/tools/pydantic-ai.md) (Python), [LangChain](../entities/tools/langchain.md) (JavaScript), or [Flowise](../entities/tools/flowise.md) and still be compatible, as long as they follow the payload and conversation-history criteria in a comprehensive developer guide, which includes sample template agents on [GitHub](../entities/tools/github.md).

He demos a Pydantic AI web research agent (using Claude 3.5 Haiku) that searches and summarizes articles, a lightweight [agentic RAG](../concepts/agentic-rag.md) flow, while [streaming real-time progress updates](../concepts/streaming-responses.md), and an n8n content-creator agent that streams each asset as it is produced. The real-time streaming of intermediate steps is emphasized as what makes the agents feel interactive. Cole frames the studio as an extension of his YouTube channel where agents he builds in videos will be immediately playable, and positions oTTomator as a one-stop community for tools, networking, and knowledge in AI. The studio also shares lineage with his other open-source work like [bolt.diy](../entities/tools/bolt-diy.md).
